# GAME OF LIFE

John Conway proposed a game with very simple rules:
1. The game runs on an infinite grid of cells
2. A cell can be dead or alive
3. A dead cell will be alive in the next generation if it has three alive cells next to it
4. An alive cell will remain alive in the next generation if it has two or three alive cells next to it

I have always found it kinda funny, and I have implemented my own version. If you want to see it, you can [click here](https://twikituki.github.io/GameOfLife/). You must press **SPACE** to start it. You can also change a cell's state by clicking on it and pause execution by pressing space again.
